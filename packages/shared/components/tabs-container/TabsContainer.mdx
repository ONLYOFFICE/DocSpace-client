### Array Items Properties

| Props     |   Type   | Required | Values | Default | Description           |
| --------- | :------: | :------: | :----: | :-----: | --------------------- |
| `id`      | `string` |    ✅    |   -    |    -    | Index of object array |
| `title`   | `string` |    ✅    |   -    |    -    | Tabs title            |
| `content` | `object` |    ✅    |   -    |    -    | Content in Tab        |
